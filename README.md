## 前言

> 各篇文章中的图片几乎都来自网络，来源繁多复杂且大多未表明出处……侵删啊，乖乖 0.0

学习一门技术或其中的知识点时，首先要了解其存在的目的：

- (**What**) 解决了实际生产中出现的什么问题？
- (**Why**) 问题产生的原因？
- (**How**) 解决问题的方法？

当对其解决方法有不理解时，递归地思考以上三问。
